# 🤖 TimeRover – Human-Following Medicine Delivery Robot

## 📌 Overview

TimeRover is an autonomous robot that follows a patient and delivers medicines. It improves medication adherence and reduces caregiver workload.

## 💡 Features

* Real-time human following (1.0m safe distance)
* Scheduled medicine dispensing
* 100% dispensing accuracy (lab tested)
* PID-based movement control
* Obstacle detection

## 🛠️ Components

| Component                   | Role                            |
| --------------------------- | ------------------------------- |
| Arduino Uno                 | Main controller (Brain)         |
| Ultrasonic Sensor (HC-SR04) | Distance measurement & tracking |
| IR Sensor                   | Obstacle detection              |
| L293D Motor Driver          | Motor control                   |
| TT Gear Motors              | Locomotion                      |
| Servo Motor                 | Medicine dispensing             |
| 18650 Li-ion Battery        | Power supply                    |


## 📊 Performance

* Tracking accuracy: ±5cm deviation
* Reaction time: 200ms
* Stable platform with zero spillage
